# 🏙️ Анализ рынка недвижимости Санкт-Петербурга

## 🔎 Описание
Аналитический проект по исследованию рынка недвижимости Санкт-Петербурга и Ленинградской области.  
Проект включает исследовательский анализ, построение ключевых метрик и визуализаций, а также выводы по динамике цен, востребованности районов и характеристикам объектов.

---

## 🎯 Цели исследования
- Исследовать участие объектов на рынке недвижимости
- Проанализировать распределения цен и площадей
- Выявить взаимосвязи между характеристиками жилья
- Сформулировать выводы на основе данных

---

## 📦 Данные
- Основные сущности: квартиры, цена, площадь, локация, этажность
- География: Санкт-Петербург и Ленинградская область

> Исходные данные используются в агрегированном виде и **не хранятся в репозитории**.  
> Хранится только код для загрузки и обработки данных.

---

## 🧠 Что было сделано
- Проведён исследовательский анализ данных (EDA)
- Построены распределения по цене, площади, этажам
- Проанализированы зависимости между ключевыми признаками
- Выявлены наиболее востребованные районы
- Построены визуализации для интерпретации результатов: https://datalens.yandex/9ccxidqey5fiv?tab=aW

## 📈 Основные результаты
- Средние цены и медианные значения для ключевых районов
- Сегментация объектов по параметрам
- Выявленные закономерности между ценой, площадью и местоположением
- Практические выводы по рынку недвижимости
